# 备考工作台日报导出设计

## 背景

首页“备考工作台”是用户每天进入系统后的执行面板：它展示目标岗位、冲刺天数、今日计划数量、薄弱题数量、跟踪题数量、下一道训练题、计划题单和弱点雷达。它偏执行层，和“备考指挥中心”的策略判断互补。

当前工作台只能在页面内使用。用户如果想把今天要做的题、薄弱分类和下一题同步到外部打卡、团队监督或离线待办，需要手动整理。为了继续强化免费、可携带、可复盘的备考体验，本次新增“备考工作台日报”导出。

## 目标

- 导出目标岗位、冲刺天数、生成日期、今日计划数、薄弱题数、跟踪题数和掌握率。
- 导出下一道建议训练题，包含分类、难度和跳转路径。
- 导出今日计划题单，包含题目、分类、难度和是否已在计划内。
- 导出弱点雷达分类，包含分类名、风险分和各状态数量。
- 空计划或空弱点时输出明确建议，不出现 `undefined`。
- 在 `StudyDashboard` 顶部操作区提供“复制日报”按钮，复制失败时下载 Markdown。

## 非目标

- 不改变 `buildDailyPlan`、`resolvePlanQuestions` 和 `weakAreasFromQuestions` 的算法。
- 不新增后端接口、账号同步或云端日报。
- 不替代更细的今日任务、健康雷达、能力地图报告。
- 不打开浏览器做视觉 QA。

## 内容结构

```markdown
# Java 后端 备考工作台日报

生成时间：2026-06-18

## 工作台概览
- 冲刺周期：21 天
- 今日计划：3 道
- 薄弱题：2 道
- 跟踪题：6 道
- 掌握率：50%

## 下一题
- HashMap 为什么线程不安全？
- 分类：Java 集合
- 难度：HARD
- 路径：/question/1

## 今日计划题单
1. HashMap 为什么线程不安全？
   - 分类：Java 集合
   - 难度：HARD
   - 状态：计划内

## 弱点雷达
1. Java 集合：5
   - 薄弱 1，道学习中 1，已掌握 0
```

## UI 设计

- 在 `StudyDashboard` 的 `.study-hero-actions` 中加入“复制日报”按钮。
- 使用 `CopyOutlined` 图标，保持和其他导出按钮一致。
- 成功提示“备考工作台日报已复制”。
- 剪贴板不可用时下载 `{目标岗位}-备考工作台日报.md`。
- 现有按钮较多，移动端继续依赖 actions 的 wrap，不新增解释性文案。

## 测试策略

- 新增 `studyDashboardReport.test.ts`：
  - 有计划和弱点时导出概览、下一题、题单和弱点雷达。
  - 空进度时输出计划生成建议，不出现 `undefined`。
- 新增 `StudyDashboard.test.tsx`：
  - 注入本地进度和热门题，点击“复制日报”写入剪贴板。
  - 剪贴板内容包含“备考工作台日报”、“工作台概览”、“今日计划题单”和目标岗位。
