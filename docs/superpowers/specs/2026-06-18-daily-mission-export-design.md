# 今日冲刺任务导出设计

## 背景

首页“今日冲刺任务”已经把复习到期、岗位能力短板、模拟面试复盘和今日计划聚合成一组优先级任务。它是用户每天打开网站后的第一行动入口，比普通题库站更接近私人备考教练。

当前问题是任务只能在页面内查看。用户如果想把当天 3 到 4 个任务同步到笔记、待办工具或群内打卡，需要手动复制每张卡片的标题、原因和入口。本阶段把任务清单导出成 Markdown，让当天行动计划可复制、可下载、可离线执行。

## 目标

- 导出今日任务标题、摘要、任务数量、任务清单和入口。
- 每个任务导出类型、指标、说明、推荐原因和行动入口。
- 空状态或低样本状态仍能导出可执行任务，不出现 `undefined`。
- 面板增加“复制任务”入口，剪贴板不可用时下载 Markdown。

## 非目标

- 不修改任务优先级、去重规则和生成算法。
- 不把任务写入后端或云端。
- 不导出完整题目答案，只导出行动清单。

## 内容结构

```markdown
# Java 后端 今日冲刺任务

生成时间：2026-06-18

## 任务概览
- 状态：今日冲刺任务
- 摘要：系统已根据复习到期、岗位短板...
- 任务数：4 个

## 任务清单
1. 先补逾期复习
   - 类型：复习
   - 指标：2 道
   - 说明：2 道题已经逾期...
   - 原因：来自智能复习排期
   - 入口：/study
```

## UI 设计

- 在任务标题右侧数字下方增加“复制任务”按钮。
- 使用 `CopyOutlined` 图标，保持和其他导出按钮一致。
- 成功提示“今日冲刺任务已复制”。
- 剪贴板失败时下载 `{岗位}-今日冲刺任务.md`。

## 测试策略

- `dailyMission.test.ts` 增加 Markdown 导出测试：
  - 有逾期复习和今日计划时导出标题、日期、概览、任务清单和入口。
  - 空进度导出启动任务，且不出现 `undefined`。
- 面板 UI 只调用已测纯函数，复制和下载逻辑与其他导出面板保持一致。
