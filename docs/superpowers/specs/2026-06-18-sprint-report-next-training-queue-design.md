# 冲刺报告下一轮训练队列设计

## 背景

面试冲刺报告已经覆盖健康度、今日任务、急救包、素材库、追问防线、能力地图、今日闭环、错因恢复和下一步行动。但报告缺少一个统一的“下一轮到底练什么”入口，用户需要在多个章节之间自己拼题单。

刚新增的下一轮训练队列已经把评分影响、复习债、面试错因、薄弱题、学习中题和今日计划融合为一个有解释的练习队列。冲刺报告应该复用这套队列，形成从诊断到训练入口的闭环。

## 目标

- 一页作战摘要增加下一轮训练动作。
- 正文增加 `## 下一轮训练队列` 章节。
- 下一步行动区增加“下一轮训练”行动线。
- 队列为空时输出可执行降级提示。

## 非目标

- 不改变冲刺报告其他章节排序和含义。
- 不新增页面组件。
- 不复制队列排序逻辑。

## 输出规则

- 正文章节展示状态、摘要、主行动和最多 8 道题。
- 每道题展示来源、行动、原因和入口。
- 已掌握高分题沿用“沉淀高分题目”入口，避免误导继续刷题。
- 导出内容不得包含 `undefined`。
