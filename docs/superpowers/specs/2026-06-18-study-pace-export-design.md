# 备考配速教练导出设计

## 背景

学习计划页的“备考配速教练”已经能根据冲刺周期、今日计划、复习债和模拟面试样本判断用户当前节奏。它比普通题库更像一个免费的私人督导：用户不只知道要刷什么题，还能知道今天为什么落后、应该先补计划还是先处理复习债。

当前问题是配速建议只能停留在页面里。用户如果要把今日节奏同步到个人笔记、群内打卡、面试前复盘文档或离线待办，需要手动抄写每日目标、已排计划、复习债、模拟样本和主行动。为了继续强化“完全免费 + 可携带 + 可复盘”的优势，本阶段为配速教练增加 Markdown 导出，让节奏诊断可以一键复制并保留行动入口。

## 目标

- 导出目标岗位、生成日期、配速等级、标题、摘要和主行动。
- 导出关键指标：今日目标、已排计划、复习债、模拟样本。
- 导出行动队列，保留动作、说明和入口。
- 空状态仍给出“进入题库建立轨迹”的下一步，且不出现 `undefined`。
- 面板提供“复制配速”按钮，剪贴板不可用时降级下载 Markdown。

## 非目标

- 不改变 `buildStudyPaceCoach` 的配速评分、等级和主行动选择算法。
- 不新增后端接口、账号体系或云端同步。
- 不导出完整题目答案、模拟面试原文或用户隐私数据。

## 内容结构

```markdown
# Java 后端 备考配速报告

生成时间：2026-06-18

## 配速概览
- 状态：今日节奏需要补齐
- 等级：需补齐
- 今日目标：6 道
- 已排计划：1 道
- 复习债：0 道
- 模拟样本：1 次
- 摘要：建议今日完成 6 道题...

## 指标明细
1. 今日目标
   - 数值：6 道
   - 说明：由冲刺周期自动换算

## 行动队列
1. 补齐今日计划
   - 说明：今日计划 1 道，建议补到 6 道。
   - 入口：/study
```

## UI 设计

- 在 `StudyPaceCoachPanel` 主行动区增加“复制配速”按钮。
- 使用 `CopyOutlined` 图标，保持和学习计划页其他导出按钮一致。
- 成功复制提示“备考配速报告已复制”。
- 剪贴板不可用时下载 `{岗位}-备考配速报告.md`。
- 按钮组支持换行，避免移动端挤压主行动文案。

## 测试策略

- `studyPaceCoach.test.ts` 增加 Markdown 导出测试：
  - 落后状态导出标题、日期、概览、指标明细、行动队列和 `/study`。
  - 空状态导出建立轨迹动作，且不出现 `undefined`。
- `StudyPaceCoachPanel.test.tsx` 增加复制测试：
  - 点击“复制配速”会调用剪贴板。
  - 复制内容包含配速报告标题、概览和行动入口。
